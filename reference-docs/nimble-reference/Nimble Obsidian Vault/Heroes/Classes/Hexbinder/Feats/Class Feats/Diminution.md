Reduce the mana cost of spells by 1 for each condition chosen. You can choose each condition only once per spell, and can't choose an effect that a spell does not have (e.g., Humble on a spell with no Range/Reach).
- **Humble.** The Range/Reach of the spell is halved.
- **Weak.** Roll with disadvantage (or enemies save with advantage).
- **Slow.** Spend 1 additional action (can't be used with Reaction spells).
