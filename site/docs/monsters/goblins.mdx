# Goblins

Goblins are small, chaotic creatures that rely on numbers and irritating abilities. All goblins share the trait **Haha, Missed Me!** — a retaliatory poke whenever an attack misses them.

## Goblin Taskmaster

**Level:** 2 | **Size:** Small | **Group:** Goblins

<StatBlock
  name="Goblin Taskmaster"
  stats={[
    { label: "HP", value: "30" },
    { label: "Armor", value: "Medium" },
    { label: "Speed", value: "6" },
    { label: "Level", value: "2" },
  ]}
/>

**Traits**
- **Haha, Missed Me!** Whenever an attack misses you, deal 1 psychic damage in return.
- **Meat Shield.** Can force other goblins to Interpose for him.

**Actions**
- **Stab.** 1d6+2 (or Shoot, Range 8). Then:
- **Get in here!** Call a goblin minion to the fight.
