# Welcome to the Nimblenomicon

An online rules reference for the **Nimble** tabletop RPG.

- ⚔️ [Actions](/rules/actions) — Attack, move, cast, assess
- 🩸 [Conditions](/rules/conditions) — Status effects and their effects
- 🧙 [Classes](/classes/) — Hero class options and features
- 🔍 [Search](/search) — Full-text search across all rules

## About Nimble

Nimble is a streamlined tabletop RPG system designed for fast, cinematic play. Heroes get 3 actions per turn, conditions are concise, and the math stays out of the way of the story.
